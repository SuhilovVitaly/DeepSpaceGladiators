# Deep Space Gladiators - Project Brief

## Overview
Deep Space Gladiators is a tactical space combat card game where pilots control ships in turn-based battles.

## Core Concept
- Turn-based tactical combat between two ships
- Card-based gameplay with maneuver and action cards
- Pilots with stamina manage card costs
- Ships have shields, structure (hull), and modules

## Target Platform
- .NET 10.0
- C# with modern language features

## Project Goals
1. Create a robust tactical combat engine
2. Support unit testing without UI dependencies
3. Clean separation of models and business logic
